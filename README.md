# 🏧 ATM Machine Simulation

A simple ATM Machine Simulation built using **HTML, CSS, and JavaScript**. This project demonstrates the fundamentals of front-end web development by simulating common ATM operations such as PIN authentication, balance checking, depositing money, withdrawing money, changing the PIN, viewing transaction history, and storing data using Local Storage.

 📌 Features

- 🔐 ATM Card Insertion
- 🔑 Secure PIN Authentication
- 💰 Check Account Balance
- ➕ Deposit Money
- ➖ Withdraw Money
- 🔄 Change ATM PIN
- 📜 Transaction History
- 💾 Local Storage Support
- 🚪 Exit ATM

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

 How It Works

 Login Page

- User inserts the ATM card.
- PIN input becomes enabled.
- User enters the correct 4-digit PIN.
- Successful login redirects to the dashboard.

Dashboard

Users can perform the following operations:

- Check Balance
- Deposit Money
- Withdraw Money
- Change PIN
- View Transaction History
- Exit ATM

Local Storage
The project stores the following information using Local Storage:

- Account Balance
- ATM PIN
- Transaction History
This allows the data to remain available even after refreshing the browser.
